# React.js with Socket.IO

This is a simple example project demonstrating how to integrate Socket.IO with React.js to build real-time web applications.

## Features

- Real-time communication between client and server using Socket.IO
- Simple chat application UI built with React.js
- Send and receive messages in real-time

## Technologies Used

- React.js
- Socket.IO
- Express.js
- Node.js

## Installation

Clone the repository:

```bash
git clone https://github.com/bobga/reactjs_socket.io.git
```
Navigate to the project directory:
```bash
cd reactjs_socket.io
```

Install dependencies:
```bash
npm install
```
Start the server:
```bash
npm start
```
Open your web browser and go to http://localhost:3000 to see the application running.

## Usage
- Enter your name and join the chat room.
- Start sending and receiving real-time messages with other users in the chat room.
